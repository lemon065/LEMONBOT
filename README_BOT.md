# LEMONBOT CBD SHOP - Telegram Bot

🍋🔫 **BOT TELEGRAM COMPLET** 🔫🍋

## Installation

```bash
pip install -r requirements.txt
python telegram_bot.py
```

## Fonctionnalités

✅ **Menu Produits** - Voir tous les produits CBD
✅ **Panier** - Gérer les achats
✅ **Jeux** - Clicker, Spin, Memory, Grow Plant
✅ **Système de Points** - Accumule des points en jouant
✅ **Commandes** - Passe des commandes facilement

## Commandes

- `/start` - Démarrer le bot
- `/menu` - Voir les produits
- `/jeux` - Jouer aux jeux
- `/panier` - Voir le panier
- `/aide` - Obtenir de l'aide

## Jeux Disponibles

🔥 **Clicker Game** - Clique 10 fois aussi vite que possible
🎡 **Lucky Spin** - Roue de fortune avec récompenses
🧠 **Memory Game** - Trouve les paires
🌱 **Grow Plant** - Cultive ta plante jusqu'à 100%
